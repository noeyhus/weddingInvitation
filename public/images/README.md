# 이미지 폴더

이 폴더에 청첩장에 사용할 이미지를 추가하세요.

## 필요한 이미지 목록:

- `hero.jpg` - 메인 히어로 섹션 배경 이미지 (권장 크기: 1920x1080px)
- `groom.jpg` - 신랑 프로필 사진 (권장 크기: 500x500px, 정사각형)
- `bride.jpg` - 신부 프로필 사진 (권장 크기: 500x500px, 정사각형)
- `gallery1.jpg` ~ `gallery6.jpg` - 갤러리 슬라이더 이미지 (권장 비율: 4:3)

## 참고사항:

- 이미지가 없어도 웹사이트가 동작합니다 (placeholder가 표시됨)
- 최적의 로딩 속도를 위해 이미지를 압축하여 사용하세요
- WebP 형식을 사용하면 더 빠른 로딩이 가능합니다
